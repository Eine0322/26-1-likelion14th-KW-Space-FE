### Web: 광운대 강의실 예약 관리 서비스 KW Space

<br/> [<img src="https://img.shields.io/badge/프로젝트 기간-2026.03.12~2026.07.20-orange?style=flat&logo=&logoColor=white" />]()

## 📍 프로젝트 개요

KW Space는 광운대학교의 온라인 강의실 예약 서비스 입니다.
기존 방식에 따라 사무실에 직접 방문하여 예약 현황을 확인 후 신청해야 되는 번거로움을 해결하고자 제작되었습니다.
모바일로 웹에 접속하여 현재 예약 현황과 신규 예약을 신청할 수 있습니다!

<br/>

## 📍 주요 기능

- 학교 이메일을 이용한 로그인
- 약도를 통한 강의실 예약 정보 시각화
- 강의실 예약 현황 조회
- 강의실 예약 신청 및 취소

<br/>
